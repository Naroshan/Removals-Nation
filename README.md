## Quick start

### 1. Generate the locations database

```bash
python scripts/locations_db.py
# → writes data/locations.json (1,700+ UK locations)
```

### 2. Build all location pages

```bash
python scripts/build_pages.py
# → writes dist/{service}/{location}/index.html for every combination
# → also rebuilds dist/locations.html
```

Options:

```bash
python scripts/build_pages.py --force              # Rebuild even existing pages
python scripts/build_pages.py --service man-and-van # One service only
python scripts/build_pages.py --dist output        # Custom output folder
python scripts/build_pages.py --data path/to.json  # Custom locations file
```

### 3. Patch the mobile hamburger menu (if needed)

If you've manually edited HTML or added new pages outside the generator:

```bash
python scripts/inject_mobile_menu.py
# → patches hamburger button + mobile menu overlay + JS into every HTML file
```

### 4. Deploy

Drag the `dist/` folder into [Netlify Drop](https://app.netlify.com/drop).  
That's it — no build step, no config.

## Adding locations

Open `scripts/locations_db.py` and add entries to `ALL_LOCATIONS`:

```python
("Shrewsbury", "Shropshire", "Shropshire", "SY1", "shrewsbury"),
#  ^name         ^region       ^county       ^pc    ^url-slug
```

Then rebuild:

```bash
python scripts/locations_db.py   # regenerates data/locations.json
python scripts/build_pages.py    # builds only new pages (skips existing)
```

---

## Adding a service

In `scripts/build_pages.py`, add to the `SERVICES` list:

```python
SERVICES = [
    ...
    ("garden-clearance", "Garden Clearance", "🌿"),  ← new
]
```

Also add a cost table entry to `COST_TABLES`:

```python
COST_TABLES["garden-clearance"] = [
    ("Small garden", "£150 – £300"),
    ("Large garden", "£300 – £600"),
]
```

Then run `python scripts/build_pages.py --service garden-clearance`.

## Formspree

All booking forms POST to `https://formspree.io/f/xkododjy`.  
Hidden fields on every form:

```html
<input type="hidden" name="_subject" value="New Removal Booking — RemovalsNation">
<input type="hidden" name="location" value="Camden">
<input type="hidden" name="service"  value="House Removals">
```

After submission, users are redirected to `/thank-you.html`.  
Log in at [formspree.io](https://formspree.io) to view submissions.
